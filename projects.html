MYA Cards集换卡牌游戏平台

项目概述

基于亚马逊云服务和阿里云服务搭建的集换式卡牌游戏平台，支持PC端、H5、App端应用， 实现卡牌认证、交换、争议等核心功能。从0开始构建整个系统框架，3个月内实现PC、H5应用上线， 2个月完成App MVP开发。

技术架构

  • • 后端：Spring Boot + MyBatis Plus + Redis
  • • 前端：Vue.js + React Native
  • • 数据库：MySQL + MongoDB
  • • 云服务：AWS + 阿里云
  • • 容器化：Docker + Kubernetes
